MacBookのスペック

CPU: Core 2 Duo 2.4GHz
L2キャッシュ: 3MB
FSB: 800MHz
メインメモリ: 2GB PC2-5300（DDR2-667）

帯域幅

メインメモリ帯域: 667(MHz) x 64(bit) x 2 / 8 = 10.67(GB/s)
FSB帯域: 800(MHz) x 64(bit) / 8 = 6.4(GB/s)
BSB帯域: 2.4(GHz) x 32(byte) = 76.8(GB/s)

キャッシュミス率

FSB帯域 : BSB帯域 = 6.4 : 76.8 = 1 : 12

a -> FSB帯域
b -> BSB帯域
x -> キャッシュミス率

b / (a(1-x) + bx) = 11
x = (a-11b) / (11(a-b)) = 1 / (11*11) = 0.008

キャッシュミス率: 0.8％
