これからMySQLを使うなら
----------

* Use MySQL 5.6+
 * InnoDB Online DDL
* Use utf8mb4 instead of utf8
 * Use utf8mb4_bin or utf8mb4_general_ci
* Use Barracuda
 * innodb_large_prefix
 * [InnoDBの制限とファイルフォーマットAntelopeとBarracudaの違い](http://blog.kamipo.net/entry/2014/12/05/235641)
