Supported Plugin Types

/*
  The allowable types of plugins
*/
#define MYSQL_UDF_PLUGIN             0  /* User-defined function        */
#define MYSQL_STORAGE_ENGINE_PLUGIN  1  /* Storage Engine               */
#define MYSQL_FTPARSER_PLUGIN        2  /* Full-text parser plugin      */
#define MYSQL_DAEMON_PLUGIN          3  /* The daemon/raw plugin type */
#define MYSQL_INFORMATION_SCHEMA_PLUGIN  4  /* The I_S plugin type */
#define MYSQL_MAX_PLUGIN_TYPE_NUM    5  /* The number of plugin types   */

UDF

* mycached (via kazuho-san)
 - The QPS (queries per second) of mycached
  is roughly 2x compared to using SQL
 - mycached can handle thousands of connections
  simultaneously

http://developer.cybozu.co.jp/kazuho/2009/08/mycached-memcac.html

STORAGE ENGINE

* Spider and VP
 - Sharding and Vertical Partitioning
* groonga
 - Fulltext search engine

FTPARSER

* mysqlftppc
 - MySQL full-text parser plugin collection
  -- bigram plugin
  -- mecab plugin
  -- space plugin
  -- suffix plugin
  -- snowball plugin

DAEMON

* HandlerSocket (via higuchi-san)
 - mycachedと同様にhandlerインタフェースをたたく
 - SQLと比べて数倍~10倍ぐらい速い
